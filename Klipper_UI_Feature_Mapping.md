# Phrozen Klipper 後台頁面功能區塊盤點與對照

本文檔彙整了由先前開發之 Klipper 後台頁面截圖，所梳理出的完整 UI 功能列表，並將截圖與對應之功能詳細說明進行了一對一的配對。

## 🛠️ 完整功能盤點清單 (含補充功能)

| 分類區塊 | 功能細項 |
| :--- | :--- |
| **儀表板 (Dashboard)** | 狀態看板 (列印清單、時間進度、開始/重印) |
| | 攝影機 (多鏡頭串流切換與滿版預覽) |
| | 移動控制 (XYZ 歸零、點動、步進解鎖) |
| | 擠出控制 (進退料、擠出機溫度) |
| | 參數微調 (速度、流量、壓力補償、擠出流量滑桿) |
| | 巨集按鈕 (自訂功能按鈕、T0-T15 / P28 等巨集) |
| | 溫度控制 & 溫度折線圖 (即時目標/當前溫度、變化圖表) |
| | *[補充]* **風扇和輸出** (模型風扇、腔溫風扇、特定 Pin 腳切換) |
| | *[補充]* **打印機限制** (加速度、轉角速度等參數即時調整) |
| **終端機 (Console)** | G-Code 終端命令列 (Log 顯示與直接發送命令) |
| **任務與檔案 (Jobs)** | 遠端 G-code 檔案管理 (列表檢視、容量大小、上傳時間) |
| | *[補充]* **Gcode 預覽** (2D 即時圖層檢視、當前進度與總層數) |

| | *[補充]* **磁盤信息** (EMMC/SD卡內部儲存空間與使用率視覺化) |

## 📸 圖片與功能詳細配對說明

### 1. 就緒狀態與歷史記錄
![就緒狀態](./images/Screenshot%202026-04-06%20111955.png)
* **對應功能：** 狀態看板 / 列印歷史任務
* **說明：** 顯示目前為機台就緒狀態，並條列先前的 `.gcode` 列印紀錄與執行狀態（包含檔名、此檔案列印耗時、是否成功）。

### 2. 攝影機多重預覽監控
![攝影機預覽](./images/Screenshot%202026-04-06%20112032.png)
* **對應功能：** 攝影機 (Webcam Stream)
* **說明：** 支援並排顯示多個攝影機視角（如圖中的 my_cam0 與 my_cam1）並動態顯示 FPS 幀數，以便完全掌握機台內部即時運作狀態。

### 3. 移動與擠出綜合控制面板
![移動控制面板](./images/Screenshot%202026-04-06%20112051.png)
* **對應功能：** 移動控制、擠出控制、巨集按鈕面板
* **說明：** 此為控制樞紐。左上位提供直觀的 X/Y/Z 歸零與手動點動位移；右上側可精確控制進退料參數；中段附帶大量自訂巨集按鈕 (如 AMS 連線、Toolhead 切替或暫停繼續)，下半段有拉桿快速修改全域列印參數之倍率。

### 4. 溫度監測與動態折線圖
![溫度動態圖表](./images/Screenshot%202026-04-06%20112145.png)
* **對應功能：** 溫度控制與監視
* **說明：** 以區塊列表形式提供噴頭 (Extruder)、熱床 (Heater Bed)、艙溫 (Chamber) 的讀數與目標設置，並於下方渲染出溫控穩定性的歷史時間變化圖表 (Thermistor Graph)。

### 5. 終端機與命令控制台
![終端控制台](./images/Screenshot%202026-04-06%20112153.png)
* **對應功能：** 終端機介面 (Console Terminal)
* **說明：** Klipper 直連溝通。可監看統背景執行的 Python 指令痕跡及 Debug 訊息，並備有 G-code 手動輸入框及常用捷徑鈕 (SEND / PAUSE / CANCEL)。

### 6. 內置任務/儲存檔案總管
![任務檔案清單](./images/Screenshot%202026-04-06%20112201.png)
* **對應功能：** G-code 檔案管理系統
* **說明：** 遠端管理 Moonraker / 機內的列印文件，涵蓋目錄切換、USB 讀取、顯示已上傳之 `.gcode` 大小、時間，並提供排序與快速搜尋。

### 7. 特殊風扇與輸出針腳控制
![風扇輸出針腳](./images/Screenshot%202026-04-06%20112241.png)
* **對應功能：** 風扇和硬體輸出控制
* **說明：** 用以單獨微調機台特殊硬體，例如模型冷卻扇 (Part Fan)、機箱排風 (Chamber Fan)轉速比，或是控制主機板上特定的 Probe / LED 開關等 GPIO 設備。

### 8. 打印機動態限制調整
![打印機性能限制](./images/Screenshot%202026-04-06%20112246.png)
* **對應功能：** 動態限制參數調整面板
* **說明：** 允許熟階使用者於機器運作時，臨時覆蓋並重定義 Klipper 核心的 `Velocity`、`Square Corner Velocity` 及 `Acceleration` 等物理極限閥值。

### 9. 即時 G-code 圖層解碼檢視
![G-code預覽](./images/Screenshot%202026-04-06%20112257.png)
* **對應功能：** 2D 即時圖層預覽 (Gcode Viewer)
* **說明：** G-Code 分析模組，利用 2D 畫布呈現特定進度處的走線與平面極限。提供目前運作在第幾層、Z高度以及總目標物件高度比。*(※ 該截圖尚未加載任何模型的實際線段渲染。)*

### 10. 熱床網格補償(Bed Mesh)曲面視覺化
![熱床網格補償曲面](./images/Screenshot%202026-04-06%20112305.png)
* **對應功能：** 熱床水平資料視覺化
* **說明：** 透過 3D 三維散點繪圖，轉換探針取樣後得到的落差值，幫助調校者一目了然得知建構底座哪邊高、哪邊低，並附上極致的公差數據標尺及色彩溫譜對應。

### 11. 系統設定與腳本文件編修
![系統文件編輯器](./images/Screenshot%202026-04-06%20112342.png)
* **對應功能：** 設定檔案管理員
* **說明：** 直接讓用戶觸及 Klipper 生態中最關鍵的 `printer.cfg`、巨集定義集、以及網路套件配置源，免去 SSH 進入修改的麻煩。

### 12. 系統錯誤與執行日誌查核
![錯誤日誌收集](./images/Screenshot%202026-04-06%20112349.png)
* **對應功能：** 服務執行系統日誌
* **說明：** 出現不可預期 Bug 或錯誤通報中斷時的後援功能，集結所有韌體 (Klippy)、服務器 (Moonraker / Fluidd) 及外掛模組的 .log 記錄，並留存壓縮歷史檔。

### 13. 硬體與主機核心綜合資訊
![系統核心信息](./images/Screenshot%202026-04-06%20112402.png)
* **對應功能：** 系統基本資訊
* **說明：** 機器的身分證，明列 CPU 處理器等級、作業系統核心環境與發行版、主機板網路 IP 分配，並提供一鍵取得完整 Log 包以及遠端重啟各項系統層級組件的安全按鍵。

### 14. 機內磁碟與儲存容量檢查
![磁碟容量檢測](./images/Screenshot%202026-04-06%20112408.png)
* **對應功能：** 主儲存空間狀態
* **說明：** 輔助性介面，協助用戶判斷機載 EMMC 或者是外掛 SD 卡空間是否已滿載，視覺化展示已用量及還能上傳多少檔案的剩餘額度。
